## Introduction:
  The online survey system is an application stored in the user server. The purpose of the system is to resolve the client to allow system users to perform tasks related to creating and/or taking a survey. Non-member users are only allowed to take anonymous surveys; non-member users are required to create an account in order to use the rest of the features from the system. Registered users are required to login into their account prior to using the functionalities of the system.
  

## Technology Stack:
  1) Java 
  2) Oracle MySQL database 
  3) Apache Web Server 
  4) PHP
  5) HTML/CSS
